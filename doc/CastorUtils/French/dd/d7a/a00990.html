Description détaillée

template<typename T>
class castor::Line3D< T >

Equation d'une droite 3D.

Auteur
Sylvain DOREMUS
Date
14/08/2010
Remarques
Une pente et une origine
x = m_slope[0] * t + m_origin[0]
y = m_slope[1] * t + m_origin[1]
z = m_slope[2] * t + m_origin[2]
