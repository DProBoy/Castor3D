template<typename T>
float castor::RangedValue< T >::percent ( ) const
inlinenoexcept
Renvoie
0.0 si m_value <= m_range.m_min.
1.0 si m_value >= m_range.m_max.
Une valeur allant de 0.0 à 1.0, selon que la valeur est plus proche de m_range.m_min ou de m_range.m_max.
