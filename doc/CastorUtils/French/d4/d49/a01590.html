Description détaillée

Représente une composante de couleur (R, V, B ou A) pour l'utilisation dans castor::RgbColour ou castor::RgbaColour.

A faire:
Remove accessors, create traits
Remarques
Définit les opérateurs de conversion en float ou uint8_t, avec les opérations correspondantes.
La valeur d'une composante de couleur est un nombre flottant compris entre 0.0 et 1.0.
