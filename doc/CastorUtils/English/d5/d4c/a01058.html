Detailed Description

template<typename Object>
class castor::FixedGrowingSizeMarkedMemoryData< Object >

Memory allocation policy. It can grow of a fixed objects count.

Author
Sylvain DOREMUS
Version
0.8.0
Date
08/01/2016
Remarks
Allocates an additional byte, marks it, to be an overview of memory leaks. Holds the memory buffers, free chunks and currently allocated objects count.
