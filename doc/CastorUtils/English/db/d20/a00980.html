Detailed Description

template<typename T>
class castor::Line3D< T >

3D line equation

Author
Sylvain DOREMUS
Date
14/08/2010
Remarks
A slope and an origin
x = m_slope[0] * t + m_origin[0]
y = m_slope[1] * t + m_origin[1]
z = m_slope[2] * t + m_origin[2]
