Description détaillée

Classe de base d'effet post rendu.

Auteur
Sylvain DOREMUS
Version
0.7.0.0
Date
20/11/2012
Remarques
Un effet post rendu est un effet appliqué après le rendu 3D et avant le rendu 2D.
Les effets post rendu sont appliqués de manière cumulative.
